2.致命的な欠陥を修正し終わったら、チャッピーと相談しながら「新しいLPのデザインの方向性」と、「今回改善したいことを1つ」決める。
3.昨日のdayXXXフォルダをコピーして、LPとREADEMEのスクショと、images内の画像ファイルを削除する。
4.フォルダ名、フッター、READMEの中身を今日の日付に変更する。templateフォルダ内のREADME.mdをコピペして差し替える
