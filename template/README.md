##毎日の詳細マニュアル
1.昨日のdayXXXフォルダの、html,css,READMEをチャッピーに共有して、致命的な欠陥がないかを確認する
2.致命的な欠陥を修正し終わったら、チャッピーと相談しながら「新しいLPのデザインの方向性」と、「今回改善したいことを1つ」決める。
3.昨日のdayXXXフォルダをコピーして、フォルダ名、タイトル、フッター、READMEの中身を今日の日付に変更して、昨日のLPとREADEMEのスクショを削除する
4.チャッピーと壁打ちしながら、想定した「デザイン」の大枠をつくる
5.デザインに合った画像をAdobeExpressの中から探す
4.チャッピーと相談しながら、今回1つ決めた「改善したいこと」を実装する
5.検証ツールを開いて、⌘+shift+Pでコマンドの検索ツールを出す
6.capture full size screenshot と打ち込んで、スクショをダウンロードする
7.README.mdに今日の成果を記載して、記載が終わったらスクショを撮る
8.ダウンロードしたLPのスクショと、README.mdのスクショを、今日のdayXXXフォルダのに移動する。形式は、LPのスクショは、「dayXXX.html.png」README.mdのスクショは「dayXXX.png」にする
9.リベッターにLPとREADMEのスクショを添付して、今日の報告のつぶやきをする。
